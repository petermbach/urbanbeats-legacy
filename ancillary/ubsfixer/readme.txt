--------------------------------------
UrbanBEATS .ubs Savefile Fixer Readme
--------------------------------------
v1.0 - 2016-01

NOTE: USE ONLY IF YOU ARE FAMILIAR WITH THE MODEL'S .UBS FILE STRUCTURE.

Simply place both ubeat_ubs_fixer.py and run_ubs_fixer.bat into the folder containing the relevant components of the ubs savefile
and run the .bat file. It will recreate the UBS file, from your modified savefiles, which should load into UrbanBEATS if parameter values
have been correctly set.
